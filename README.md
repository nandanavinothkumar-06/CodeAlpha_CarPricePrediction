# Car Price Prediction using Machine Learning

## Overview
This project focuses on predicting the selling price of used cars based on various features such as present price, fuel type, transmission type, kilometers driven, and ownership using Machine Learning regression techniques.

The project demonstrates:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Categorical variable encoding
- Regression modeling
- Model evaluation and prediction

---

## Objective
The objective of this project is to build a Machine Learning model capable of predicting car selling prices based on multiple car-related features.

## Exploratory Data Analysis
The project includes:
- Correlation Heatmap
- Selling Price Distribution
- Present Price vs Selling Price Analysis

### Key Observations
- Present Price showed the strongest positive correlation with Selling Price.
- Kms Driven had a negative relationship with Selling Price.
- Fuel type and transmission influenced the resale value of cars.
